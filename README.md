# projet_c_pgcd
Le projet a donc pour objectif de réaliser une "bibliothèque" de fonctions permettant la manipulation d’entiers arbitrairement grands, en binaire. Une définition simple de ce qu’est un "grand entier" : c’est un entier qui ne peut être exactement représenté sur 32 bits ou sur 64 bits, ni même sur 128 bits.
